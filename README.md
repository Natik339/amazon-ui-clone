Amazon Clone

A responsive frontend clone of the Amazon homepage built using pure HTML and CSS without any frontend frameworks or libraries like Bootstrap.

🚀 Features
Responsive Navigation Bar
Amazon-style Hero Section
Product Cards Layout
Footer Section
Hover Effects
Clean UI Design
Pure HTML5 & CSS3
Responsive Design for Smaller Screens

🛠️ Tech Stack
HTML5
CSS3
Font Awesome

## ⚠️ Note
**This project is best viewed on a laptop/desktop screen. Full responsiveness for all devices has not been completely implemented yet, as this project focuses mainly on basic HTML and CSS concepts.
